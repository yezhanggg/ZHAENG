# ZHAENG Poster Designer

A powerful, creative poster design tool with bold text styles and real-time customization.

## Features

### Image Upload
- Click or drag & drop to upload background image
- Supports all standard image formats (JPG, PNG, WebP, etc.)
- Real-time preview

### 12 Creative Text Styles

1. **IMPACT** - Ultra bold condensed with stroke
2. **NEON GLOW** - Cyberpunk glowing effect
3. **GRAFFITI** - Street art spray paint style
4. **RETRO WAVE** - 80s chrome gradient (pink/cyan)
5. **BRUTALIST** - Raw geometric monospace
6. **3D POP** - Comic book extruded effect
7. **GLITCH** - RGB chromatic aberration
8. **STENCIL** - Military/industrial cut-out
9. **OUTLINE** - Hollow letters with thick stroke
10. **SHADOW DROP** - Heavy lifted shadow effect
11. **VINTAGE** - Classic poster advertising
12. **MARKER** - Hand-drawn thick marker

### Customization Options

- **Text Input**: Multi-line support, type anything
- **Size**: Adjustable from 20px to 200px
- **Color**: Full color picker
- **Position**: Top, Middle, or Bottom placement
- **Rotation**: -45° to +45° angle adjustment

### Export

- High-quality PNG download
- Preserves original image resolution
- Professional output ready for printing

## How to Use

1. **Upload Image**: Click upload area or drag & drop your background image
2. **Enter Text**: Type your text in the input field (supports multiple lines)
3. **Choose Style**: Select from 12 creative text styles
4. **Customize**: Adjust size, color, position, and rotation
5. **Save**: Download your finished poster as PNG

## Technical Details

- Canvas-based rendering for high quality
- Real-time preview with instant updates
- Responsive design works on mobile and desktop
- Clean glassmorphism UI matching ZHAENG aesthetic

## Tips

- Use high-resolution images for best results
- Bold, short text works best for impact
- Experiment with different styles for your message
- Try rotation for dynamic layouts
- Layer multiple text elements by saving and re-uploading

## Browser Support

Works on all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile iOS and Android browsers
