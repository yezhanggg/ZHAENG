# Image Scrambler - User Guide

Welcome to the Image Scrambler! This tool transforms your images into beautiful chaos by breaking them into hundreds or thousands of pieces and shuffling them.

## Quick Start

### 1. Upload Your Image
- **Click** the upload area or **drag and drop** an image
- Supports: JPG, PNG, GIF, WebP
- Any size works (larger images = more detail)

### 2. Choose Your Grid Size
Select from the dropdown:
- **500 pieces**: Good for quick scrambles
- **1000 pieces**: Default, balanced chaos
- **2000 pieces**: Maximum scrambling
- **Custom**: Create your own grid (5-100 rows/columns)

### 3. Scramble!
Click the **Scramble** button and watch your image transform into organized chaos!

## Tips & Tricks

### For Best Results

**Portrait Photos**:
- Use 1000-2000 pieces for maximum scrambling
- Faces become unrecognizable but colors remain

**Landscapes**:
- 500-1000 pieces preserve some recognizability
- Sky and terrain create interesting patterns

**Text/Documents**:
- Higher piece counts (2000+) completely obscure text
- Great for privacy while keeping document aesthetics

**Abstract Images**:
- Any piece count works
- Creates layered abstract effects

### Performance Tips

- **Large images**: May take a moment to process
- **High piece counts**: Custom grids above 100√ó100 may be slow
- **Mobile devices**: Stick to 500-1000 pieces for best performance

### Artistic Techniques

1. **Subtle Scramble**: Use 500 pieces, keep grid visible
2. **Complete Chaos**: Use 2000+ pieces, scramble multiple times
3. **Controlled Randomness**: Custom grid with specific ratios
4. **Mosaic Effect**: Enable grid overlay, use 1000 pieces

## Troubleshooting

### Image won't upload
- Check file format (JPG, PNG, GIF, WebP only)
- Try a smaller file size
- Ensure browser supports File API

### Scramble is slow
- Reduce grid size
- Use smaller images
- Close other browser tabs

### Download doesn't work
- Check browser download permissions
- Ensure pop-ups aren't blocked
- Try right-click ‚Üí Save Image

### Grid doesn't show
- Ensure checkbox is checked
- Try scrambling/resetting
- Refresh the page if stuck

## Privacy & Security

- **All processing is local**: Images never leave your browser
- **No uploads to servers**: Everything happens on your device
- **No data storage**: Nothing is saved when you close the page
- **Safe for sensitive content**: Use for privacy-preserving scrambles

4. **Double Scramble**: Download a scrambled image, then scramble it again!
